## Installation

```bash
pnpm dlx shadcn@latest add @fabric-ui/page
```

## Usage

```tsx
import { Page } from "@repo/page";
```

```tsx
<Page title="Products" description="Manage your products and inventory.">
  {/* Page content */}
</Page>
```
