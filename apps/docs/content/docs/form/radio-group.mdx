## Installation

```bash
pnpm dlx shadcn@latest add @fabric-ui/radio-group
```

## Usage

```tsx
import { RadioGroup } from "@/components/fabric-ui/radio-group";
```

```tsx
<RadioGroup
  label="Options"
  description="Select your preferred option"
  defaultValue="option1"
  items={[
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
    { value: "option3", label: "Option 3" },
  ]}
/>
```

## Features

- **Single Selection**: Only one option can be selected at a time
- **Controlled & Uncontrolled**: Works in both controlled and uncontrolled modes
- **Accessible**: Proper fieldset semantics with label and description support
- **Keyboard Navigation**: Full keyboard support for option selection

## Best Practices

- Use clear labels for each radio option
- Use the `label` and `description` props to provide context
- Add descriptions to individual items when additional context is needed
- Use radio groups when users need to select exactly one option from a list
