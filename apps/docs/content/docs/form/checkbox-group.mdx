## Installation

```bash
pnpm dlx shadcn@latest add @fabric-ui/checkbox-group
```

## Usage

```tsx
import { CheckboxGroup } from "@/components/fabric-ui/checkbox-group";
```

```tsx
<CheckboxGroup
  label="Options"
  description="Select your preferred options"
  defaultValue={["option1"]}
  items={[
    { value: "option1", label: "Option 1" },
    { value: "option2", label: "Option 2" },
    { value: "option3", label: "Option 3" },
  ]}
/>
```

## Examples

### Parent Checkbox

Use the `parent` prop to create a checkbox that controls all child checkboxes. The parent checkbox will show an indeterminate state when some but not all children are selected.

## Features

- **Collective State Management**: Manages the state of multiple checkboxes as a group
- **Parent Checkbox**: Support for a parent checkbox that can select/deselect all children
- **Indeterminate State**: Automatically shows indeterminate state on parent when partially selected
- **Controlled & Uncontrolled**: Works in both controlled and uncontrolled modes
- **Accessible**: Proper fieldset semantics with label and description support

## Best Practices

- Use clear labels for each checkbox option
- Use the `label` and `description` props to provide context
- Use the parent checkbox pattern for "select all" functionality
- Add descriptions to individual items when additional context is needed
